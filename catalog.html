Browse and install community scripts. Compatibility data is community-curated — your mileage may vary.

about compatibility: norns-web runs Lua scripts in the browser using the Web Audio, Web MIDI, and softcut APIs. Scripts that require SuperCollider engines cannot produce audio in norns-web. partial = core functionality works; incompatible = no audio output without engine. Install requires running norns-web via a local HTTP server (e.g. npx serve).
